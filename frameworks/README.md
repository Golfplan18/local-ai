# Framework Library

This directory contains frameworks used by the Local AI First Boot system.

## Structure

- `book/` — Frameworks shipped with the book (cloned from repository when available)
- `user/` — User-created frameworks
- `framework-registry.md` — Index of all available frameworks

## Repository

The book's framework repository will be cloned here when it is published and its URL is established.

Expected URL: [to be determined upon publication]

To clone when available:
```
git clone [REPOSITORY_URL] ~/local-ai/frameworks/book
```

To update after cloning:
```
cd ~/local-ai/frameworks/book && git pull
```

## Available Now

The framework registry (framework-registry.md) documents all frameworks included with this system.
